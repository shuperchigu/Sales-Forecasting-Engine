# рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃўрЃарЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃў

# ­ЪЊѕ Predictive Inventory & Order Automation Engine

### **рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЏрЃўрЃЏрЃЮрЃ«рЃўрЃџрЃЋрЃљ (Executive Summary)**

рЃћрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃљрЃарЃўрЃА **Advanced Analytics**-рЃўрЃА рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃ░рЃўрЃЉрЃарЃўрЃЊрЃБрЃџ ML рЃЏрЃЮрЃЊрЃћрЃџрЃћрЃЉрЃА рЃўрЃюрЃЋрЃћрЃюрЃбрЃљрЃарЃўрЃА рЃќрЃБрЃАрЃбрЃў рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃўрЃарЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃерЃћрЃЎрЃЋрЃћрЃЌрЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃўрЃА рЃАрЃарЃБрЃџрЃў рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃўрЃќрЃљрЃфрЃўрЃўрЃАрЃЌрЃЋрЃўрЃА.

рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃдрЃўрЃарЃћрЃЉрЃБрЃџрЃћрЃЉрЃљ: **рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃњрЃљрЃЊрЃљрЃЦрЃфрЃћрЃЋрЃљ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃа, рЃцрЃўрЃюрЃљрЃюрЃАрЃБрЃа рЃњрЃљрЃЊрЃљрЃгрЃДрЃЋрЃћрЃбрЃўрЃџрЃћрЃЉрЃћрЃЉрЃљрЃЊ**.

---

## ­ЪДа рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃ«рЃћрЃЊрЃЋрЃљ рЃЊрЃљ ML рЃџрЃЮрЃњрЃўрЃЎрЃљ

рЃћрЃА рЃАрЃбрЃарЃљрЃбрЃћрЃњрЃўрЃљ рЃБрЃюрЃўрЃЋрЃћрЃарЃАрЃљрЃџрЃБрЃарЃљрЃЊ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃа рЃЎрЃЮрЃЏрЃърЃљрЃюрЃўрЃљрЃерЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃюрЃћрЃЊрЃџрЃћрЃБрЃџрЃА рЃљрЃю рЃЏрЃљрЃарЃљрЃњрЃћрЃЉрЃА рЃЏрЃљрЃарЃЌрЃљрЃЋрЃА.

| рЃћрЃбрЃљрЃърЃў | рЃцрЃЮрЃЎрЃБрЃАрЃў | рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃБрЃарЃў рЃдрЃўрЃарЃћрЃЉрЃБрЃџрЃћрЃЉрЃљ |
| :--- | :--- | :--- |
| **1. рЃАрЃбрЃарЃљрЃбрЃћрЃњрЃўрЃБрЃџрЃў KPI-рЃћрЃЉрЃў** | **рЃцрЃўрЃюрЃљрЃюрЃАрЃБрЃарЃў рЃЏрЃљрЃарЃЌрЃЋрЃљ рЃЊрЃљ рЃарЃўрЃАрЃЎрЃў** | рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў рЃћрЃцрЃБрЃФрЃюрЃћрЃЉрЃљ **ADD (Average Daily Demand), DSI, GMROI** рЃЊрЃљ рЃљрЃарЃљ рЃБрЃЉрЃарЃљрЃџрЃЮ рЃњрЃљрЃДрЃўрЃЊрЃЋрЃћрЃЉрЃА (рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃўрЃА рЃњрЃљрЃЊрЃљрЃГрЃарЃљ: "рЃЌрЃБ рЃЏрЃљрЃарЃљрЃњрЃерЃў рЃљрЃа рЃўрЃДрЃЮ, рЃњрЃљрЃДрЃўрЃЊрЃЋрЃљ рЃљрЃа рЃЏрЃЮрЃ«рЃЊрЃћрЃЉрЃЮрЃЊрЃљ"). |
| **2. рЃ░рЃўрЃЉрЃарЃўрЃЊрЃБрЃџрЃў ML рЃЏрЃЮрЃЊрЃћрЃџрЃў** | **Advanced Analytics** | **FB Prophet** (рЃЊрЃарЃЮрЃўрЃА рЃАрЃћрЃарЃўрЃћрЃЉрЃўрЃА рЃбрЃарЃћрЃюрЃЊрЃћрЃЉрЃў) рЃўрЃюрЃбрЃћрЃњрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў **XGBoost-рЃЌрЃљрЃю** (рЃарЃћрЃњрЃарЃћрЃАрЃЮрЃарЃћрЃЉрЃў, рЃљрЃЦрЃфрЃўрЃћрЃЉрЃў, рЃњрЃљрЃарЃћ рЃцрЃљрЃЦрЃбрЃЮрЃарЃћрЃЉрЃў). |
| **3. Logistic Growth** | **рЃЉрЃљрЃќрЃарЃўрЃА рЃќрЃарЃЊрЃўрЃА рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃарЃћрЃЉрЃљ** | рЃърЃарЃЮрЃЊрЃБрЃЦрЃбрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃљрЃГрЃљрЃарЃЉрЃћрЃЉрЃћрЃю рЃќрЃарЃЊрЃўрЃА рЃЉрЃљрЃарЃўрЃћрЃарЃА, рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ **Logistic Growth Model** рЃЉрЃљрЃќрЃарЃўрЃА рЃџрЃўрЃЏрЃўрЃбрЃўрЃА (Cap) рЃњрЃљрЃЌрЃЋрЃљрЃџрЃўрЃАрЃгрЃўрЃюрЃћрЃЉрЃўрЃЌ. |
| **4. рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў Output** | **рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮ рЃЉрЃўрЃќрЃюрЃћрЃА рЃњрЃљрЃЋрЃџрЃћрЃюрЃљ** | рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃў рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃЦрЃЏрЃюрЃўрЃА **рЃерЃћрЃАрЃљрЃЎрЃЋрЃћрЃЌ рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃљрЃА** рЃЊрЃљ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃўрЃгрЃћрЃарЃћрЃЉрЃљ рЃЉрЃљрЃќрЃљрЃерЃў рЃљрЃю Excel-рЃерЃў, рЃарЃљрЃф рЃљрЃБрЃЦрЃЏрЃћрЃЉрЃА рЃ«рЃћрЃџрЃўрЃЌ рЃерЃћрЃЎрЃЋрЃћрЃЌрЃћрЃЉрЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮрЃћрЃЉрЃљрЃА. |

---

## ­ЪЏа№ИЈ рЃбрЃћрЃЦрЃюрЃўрЃЎрЃБрЃарЃў рЃАрЃбрЃћрЃЎрЃў рЃЊрЃљ рЃўрЃюрЃЪрЃўрЃюрЃарЃБрЃџрЃў рЃАрЃўрЃЏрЃДрЃљрЃарЃћ

* **ML Frameworks:** Facebook Prophet, XGBoost (implicit), Pandas
* **Engineering:** Python, SQLAlchemy, Psycopg2
* **Architecture:** Modular design (`utils.py`, `config.py`, `db.py`)
* **Deployment Target:** AWS (EC2/Airflow Orchestration).

---

## рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ

- `main.py`: рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃњрЃљрЃЏрЃерЃЋрЃћрЃЉрЃў рЃАрЃЎрЃарЃўрЃърЃбрЃў.
- `config.py`: рЃДрЃЋрЃћрЃџрЃљ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ рЃЊрЃљ рЃ░рЃўрЃърЃћрЃарЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў.
- `prophecy.py`: Prophet-рЃўрЃА рЃърЃарЃЮрЃњрЃюрЃЮрЃќрЃўрЃарЃћрЃЉрЃўрЃА рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃџрЃЮрЃњрЃўрЃЎрЃљ.
- `evaluate.py`: рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃА рЃАрЃўрЃќрЃБрЃАрЃбрЃўрЃА рЃерЃћрЃцрЃљрЃАрЃћрЃЉрЃўрЃА рЃџрЃЮрЃњрЃўрЃЎрЃљ.
- `optimize.py`: рЃЉрЃўрЃБрЃ»рЃћрЃбрЃўрЃА рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃЏрЃЮрЃЊрЃБрЃџрЃў.
- `utils.py`: рЃЊрЃљрЃЏрЃ«рЃЏрЃљрЃарЃћ рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў (рЃЏрЃљрЃњ. рЃЏрЃЮрЃЊрЃћрЃџрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ).
- `db.py`: рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃљрЃАрЃЌрЃљрЃю рЃЎрЃљрЃЋрЃерЃўрЃарЃў.
- `query/LoadData.py`: рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЕрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃўрЃА рЃџрЃЮрЃњрЃўрЃЎрЃљ.
- `holidays.py`: рЃЊрЃдрЃћрЃАрЃљрЃАрЃгрЃљрЃБрЃџрЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљ.
- `requirements.txt`: рЃАрЃљрЃГрЃўрЃарЃЮ рЃърЃўрЃЌрЃЮрЃюрЃўрЃА рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃў.
- `.env.example`: рЃњрЃљрЃарЃћрЃЏрЃЮрЃА рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃўрЃА рЃерЃљрЃЉрЃџрЃЮрЃюрЃў.




## рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ рЃЊрЃљ рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ

1.  **рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ рЃЊрЃљ рЃњрЃљрЃљрЃљрЃЦрЃбрЃўрЃБрЃарЃћрЃЌ рЃЋрЃўрЃарЃбрЃБрЃљрЃџрЃБрЃарЃў рЃњрЃљрЃарЃћрЃЏрЃЮ (рЃарЃћрЃЎрЃЮрЃЏрЃћрЃюрЃЊрЃўрЃарЃћрЃЉрЃБрЃџрЃўрЃљ):**
    ```bash
    python -m venv venv
    source venv/bin/activate  # macOS/Linux
    .\venv\Scripts\activate    # Windows
    ```

2.  **рЃЊрЃљрЃљрЃДрЃћрЃюрЃћрЃЌ рЃАрЃљрЃГрЃўрЃарЃЮ рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃў:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **рЃерЃћрЃЦрЃЏрЃћрЃюрЃўрЃЌ `.env` рЃцрЃљрЃўрЃџрЃў:**
    - рЃњрЃљрЃЊрЃљрЃљрЃарЃЦрЃЋрЃўрЃЌ `.env.example` рЃцрЃљрЃўрЃџрЃА рЃАрЃљрЃ«рЃћрЃџрЃў рЃЊрЃљ рЃЊрЃљрЃљрЃарЃЦрЃЋрЃўрЃЌ `.env`.
    - рЃерЃћрЃљрЃЋрЃАрЃћрЃЌ рЃцрЃљрЃўрЃџрЃў рЃЌрЃЦрЃЋрЃћрЃюрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃўрЃА рЃарЃћрЃљрЃџрЃБрЃарЃў рЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃўрЃЌ.
    ```
    DB_USER="your_db_user"
    DB_PASSWORD="your_db_password"
    DB_HOST="your_db_host"
    DB_PORT="5432"
    DB_NAME="your_db_name"
    ```

4.  **рЃњрЃљрЃБрЃерЃЋрЃўрЃЌ рЃърЃарЃЮрЃћрЃЦрЃбрЃў:**
    ```bash
    python main.py
    ```


рЃърЃарЃЮрЃфрЃћрЃАрЃўрЃА рЃЊрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ, рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃў рЃерЃћрЃўрЃюрЃљрЃ«рЃћрЃЉрЃљ `output/` рЃцрЃЮрЃџрЃЊрЃћрЃарЃерЃў.
