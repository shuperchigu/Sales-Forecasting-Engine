# გაყიდვების პროგნოზირების პროექტი

ეს პროექტი იყენებს Facebook Prophet-ს, რათა იწინასწარმეტყველოს პროდუქტების საშუალო დღიური მოთხოვნა (Average Daily Demand) e-commerce პლატფორმისთვის.

## პროექტის სტრუქტურა

- `main.py`: მთავარი გამშვები სკრიპტი.
- `config.py`: ყველა კონფიგურაცია და ჰიპერპარამეტრი.
- `prophecy.py`: Prophet-ის პროგნოზირების ძირითადი ლოგიკა.
- `evaluate.py`: მოდელის სიზუსტის შეფასების ლოგიკა.
- `optimize.py`: ბიუჯეტის ოპტიმიზაციის მოდული.
- `utils.py`: დამხმარე ფუნქციები (მაგ. მოდელის შექმნა).
- `db.py`: მონაცემთა ბაზასთან კავშირი.
- `query/LoadData.py`: მონაცემების ჩატვირთვის ლოგიკა.
- `holidays.py`: დღესასწაულების განსაზღვრა.
- `requirements.txt`: საჭირო პითონის პაკეტები.
- `.env.example`: გარემოს ცვლადების შაბლონი.

## ინსტალაცია და გაშვება

1.  **შექმენით და გაააქტიურეთ ვირტუალური გარემო (რეკომენდირებულია):**
    ```bash
    python -m venv venv
    source venv/bin/activate  # macOS/Linux
    .\venv\Scripts\activate    # Windows
    ```

2.  **დააყენეთ საჭირო პაკეტები:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **შექმენით `.env` ფაილი:**
    - გადაარქვით `.env.example` ფაილს სახელი და დაარქვით `.env`.
    - შეავსეთ ფაილი თქვენი მონაცემთა ბაზის რეალური პარამეტრებით.
    ```
    DB_USER="your_db_user"
    DB_PASSWORD="your_db_password"
    DB_HOST="your_db_host"
    DB_PORT="5432"
    DB_NAME="your_db_name"
    ```

4.  **გაუშვით პროექტი:**
    ```bash
    python main.py
    ```

პროცესის დასრულების შემდეგ, შედეგები შეინახება `output/` ფოლდერში.
